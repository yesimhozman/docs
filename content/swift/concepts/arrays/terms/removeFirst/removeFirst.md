The **`.removeFirst()`** method removes the first element in an array.

## Syntax

```pseudo
arrayName.removeFirst()
```

## Example

```swift
var topBabyNames = ["Sophia", "Liam", "Riley", "Jackson", "Olivia", "Noah"]

topBabyNames.removeFirst()
print(topBabyNames)
```

This will output:

```shell
["Liam", "Riley", "Jackson", "Olivia", "Noah"]
```
