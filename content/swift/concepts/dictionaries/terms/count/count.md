The **`.count`** property returns an integer that represents how many key-value pairs are in a dictionary.

## Syntax

```pseudo
dictionaryInstance.count
```

## Example

```swift
var fruitStand = [
  "Apples": 12,
  "Bananas": 20,
  "Oranges", 17
]

print(fruitStand.count)
// Output: 3
```
